--- ETAPA 1 (HTML: A BASE DO FRONTEND) ---

    - <div/> criação de divs, para separar as camadas da página que iremos atuar

        - <aside/> menu lateral de informações do github
            - <a/> seleção de imagens e links para a página
        - <p/> parágrafo da página

    - (class) atributo do comando
    - (alt) informar do que se trata o códgio


--- ETAPA 2 (CSS: ESTILIZANDO A PÁGINA E COMPLETANDO A ESTRUTURA) ---

    - <link/> Conseguir juntar o arquivo css com a página html
        - arquivo: (reset.css) Resetar estilos colocados pelo próprio navegador
        - arquivo: (estilo.css) Estilizar a página com os comandos css desse arquivo
        - arquivo: (responsivo.css) Adaptar a página aos aparelhos, ex: Celular,Notebook,Tv

--- ETAPA 3 (JAVASCRIPT)

    - <script/> Linkar o aquivo html com o js
        - arquivo: (index.js) Programação de JAVASCRIPT com animação na página em relação aos botões
